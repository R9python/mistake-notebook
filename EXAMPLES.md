# 使用示例

本文档提供详细的使用示例，帮助你快速上手初中数学错题本应用。

## 场景 1：第一次使用

### 步骤 1：安装和配置

```bash
# 进入项目目录
cd /home/zhy/mycode/mistake-notebook

# 运行自动安装脚本（Linux/Mac）
./install.sh

# 或者手动安装
pip install -r requirements.txt
cp .env.example .env
```

### 步骤 2：配置 API 密钥

编辑 `.env` 文件：

```bash
ANTHROPIC_API_KEY=sk-ant-api03-xxxxx  # 替换为你的真实密钥
FLASK_SECRET_KEY=my-secret-key-12345
DATABASE_PATH=data/mistakes.db
UPLOAD_FOLDER=static/uploads
MAX_CONTENT_LENGTH=5242880
```

### 步骤 3：启动应用

```bash
python app.py
```

访问 `http://localhost:5000`，你会看到首页。

## 场景 2：上传第一道错题

### 操作步骤

1. 点击导航栏的"上传错题"
2. 选择年级：初一
3. 准备一张数学错题图片（例如：一元一次方程题目）
4. 拖拽图片到上传区域，或点击选择文件
5. 点击"开始上传并识别"

### 预期结果

- AI 自动识别题目文字
- 自动分类知识点（例如：一元一次方程）
- 自动评估难度（easy/medium/hard）
- 显示识别结果，可点击"查看详情"

### 示例输出

```
✓ 上传成功
识别的题目：解方程：2x + 5 = 13
知识点：一元一次方程
难度：easy
```

## 场景 3：查看错题详情并生成解析

### 操作步骤

1. 在首页或错题列表中，点击错题卡片
2. 进入错题详情页
3. 查看原图和识别的题目文字
4. 点击"生成解析"按钮

### 预期结果

AI 生成详细解析，包括：

```markdown
## 题目分析
本题考查一元一次方程的解法，属于基础题型。

## 解题步骤
1. 移项：2x = 13 - 5
2. 化简：2x = 8
3. 系数化为1：x = 4

## 正确答案
x = 4

## 常见错误
- 移项时忘记变号
- 计算错误

## 解题技巧
- 先移项，再合并同类项
- 最后系数化为1
- 记得验算
```

## 常见问题解答

### Q1: 图片识别不准确怎么办？

**A**:
- 确保图片清晰，文字可读
- 在错题详情页手动修正题目文字
- 重新生成解析

### Q2: 如何删除错误上传的图片？

**A**:
- 在错题详情页点击"删除"按钮
- 或在错题列表页勾选后批量删除

### Q3: API 调用失败怎么办？

**A**:
- 检查网络连接
- 确认 API 密钥正确
- 查看 API 配额是否充足
- 应用会自动重试 3 次

### Q4: 如何备份数据？

**A**:
```bash
# 备份数据库
cp data/mistakes.db data/mistakes_backup_$(date +%Y%m%d).db

# 备份图片
tar -czf uploads_backup_$(date +%Y%m%d).tar.gz static/uploads/
```

### Q5: 可以多人使用吗？

**A**:
- 当前版本是单用户设计
- 可以在局域网内共享使用
- 未来版本会支持多用户

## 学习建议

### 每日使用流程

1. **放学后**：上传当天的错题（5-10 分钟）
2. **晚自习**：查看 AI 解析，理解错误原因（15-20 分钟）
3. **周末**：生成分析报告，针对性复习（30-60 分钟）

### 复习策略

- **第 1 天**：上传错题，查看解析
- **第 3 天**：第一次复习，记录复习
- **第 7 天**：第二次复习
- **第 15 天**：第三次复习
- **第 30 天**：最后复习，标记已掌握

### 目标设定

- **短期目标**：每周减少 5 道未掌握错题
- **中期目标**：一个月内掌握 80% 的错题
- **长期目标**：期末考试前清空所有未掌握错题
