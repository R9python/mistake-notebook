🚀 快速开始
═══════════════════════════════════════════════════════════

1. 安装依赖
   pip install -r requirements.txt

2. 配置 API 密钥
   cp .env.example .env
   # 编辑 .env，填入 ANTHROPIC_API_KEY

3. 启动应用
   python app.py
   # 访问 http://localhost:5000
